API v2

Documentación API

Integra NextBoost en tu propio panel, bot o aplicación. Compatible con el estándar de los principales panels SMM.

Introducción

La API de NextBoost usa el formato estándar de los panels SMM. Todas las solicitudes son POST y las respuestas están en JSON.

URL base: https://nextboost.io/api/v2
Método: POST
Formato: application/x-www-form-urlencoded
ℹ️ Obtén tu API key desde tu panel de usuario → sección API.

Autenticación

Incluye tu key en cada solicitud.

ParámetroDescripción
keyTu API key de NextBoost
actionLa acción a ejecutar

Listar servicios

action=services
ParámetroDescripción
keyTu API key
actionservices

Respuesta

[ { "service": 1, "name": "Instagram - Seguidores Reales", "type": "Default", "category": "Instagram - Seguidores", "rate": "0.90", "min": "100", "max": "50000", "refill": true, "cancel": false } ]

Crear pedido

action=add
ParámetroRequeridoDescripción
keyReq.Tu API key
actionReq.add
serviceReq.ID del servicio
linkReq.URL del perfil o publicación
quantityReq.Cantidad
runs—Runs para drip-feed
interval—Intervalo en minutos (drip-feed)

Respuesta

{ "order": 23501 }

Estado de pedido(s)

action=status
ParámetroDescripción
keyTu API key
actionstatus
orderID de un pedido
ordersIDs separados por coma (hasta 100)

Respuesta (un pedido)

{ "charge": "0.27819", "start_count": "3572", "status": "Partial", "remains": "157", "currency": "USD" }

Estados posibles

Pending In progress Active Completed Partial Canceled

Solicitar refill

action=refill

Disponible solo para servicios con refill: true.

ParámetroDescripción
keyTu API key
actionrefill
ordersIDs separados por coma
[ { "order": 1, "refill": 1 }, { "order": 2, "refill": 2 } ]

Cancelar pedidos

action=cancel

Solo disponible para servicios con cancel: true.

{ "key": "TU_API_KEY", "action": "cancel", "orders": "9,15,23" }

Saldo de cuenta

action=balance
{ "balance": "100.84292", "currency": "USD" }

Errores

Los errores retornan un objeto con campo error:

{ "error": "Incorrect API key" }
ErrorDescripción
Incorrect API keyLa clave no existe o está inactiva
Insufficient balanceSaldo insuficiente para el pedido
Invalid serviceEl servicio no existe
Incorrect order IDEl pedido no existe o no te pertenece
Quantity out of rangeFuera del rango min/max del servicio
